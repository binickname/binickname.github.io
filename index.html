Yelp Data Set - Determining if can predict rating stars based on user text review

Michael Pham

Introduction

Yelp (http://www.yelp.com/) is a User Reviews and Recommendations of Top Restaurants, Shopping, Nightlife, Entertainment, Services... application. One of important feature Yelp provides is to allow users who uses to interact, contribute their ideas, feedbacks, thinkings via their ratings and reviewings. Normally, if users are interesting in the restaurants, services...they can write the reviews and make the rating.

My goal in this project is to answer the question: Can we predict the rating stars of user by just based on review text? If the result is good and applied, this would be a nice function for Yelp app users.

Methods and Data

The dataset provided here is part of the Yelp Dataset Challenge and can be found at: http://www.yelp.com/dataset_challenge. In scope of this project, we just focused on the yelp_academic_dataset_review.json file.

The detail of methods used in this project:

  • Loading the dataset and transform data
  • Using the R Text Mining Package (tm) to process the text, extract features by some standard steps such as: remove punctuations, stop words, stemming, remove rare words...
  • Splitting data into 70% training set and 30% test set
  • Apply ML Algorithms: CART(Classification And Regression Tree), RandomForest to fit the model
  • Train data and evaluate model results on the test data

Discussion

From the results above, we can see it's possible to predict the rating stars based on the text review even the accuracy is not high. There were some problem with current approach in feature extraction. It requires better hardware (CPU & Memory) to process a large amount of textual data.
The future work will be to improve the feature extraction by using more different approaches such as: Bigram, Trigram, Latent Dirichlet Allocation (LDA)...then compare for the better results. Also, performance for the large dataset could be investigated to improve.
